XMLHttpRequest

Système de requête AJAX de javascript. 

var req = new XMLHttpRequest();
req.open('GET', 'URL', false);
req.send(null);


if (req.status){
   alert('OK');
} else {
   alert(req.responseText);
}

Nouvelle manière de fetcher des ressources sur le réseau.
